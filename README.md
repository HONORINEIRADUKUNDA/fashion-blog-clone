# Fashion Blog Clone

This is a custom-built clone of the W3Schools Fashion Blog Template. I built this project entirely from scratch to practice my front-end web development skills.

## Objectives Completed
* Structured a webpage correctly using semantic HTML tags.
* Replicated a complex, two-column layout using **CSS Flexbox**.
* Applied custom typography (Oswald and Open Sans via Google Fonts), spacing, and box-shadows.
* Implemented a fully responsive web design so the layout adapts and stacks perfectly on mobile screens.

## Built With
* **HTML5** (100% custom structure)
* **CSS3** (100% vanilla CSS, no frameworks used)

## Project Notes
This project was built following a strict "Golden Rule": no CSS frameworks (like Bootstrap or Tailwind) or pre-made templates were allowed. All code was written manually.
